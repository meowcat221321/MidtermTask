Coding

For our project, we mostly worked with physical objects and very little technology, but I worked on the lighting system for shooting the video.

We used a cricket board and coded it to be able to shuffle through different modes of light

The following code was used for our lighting system:

 input.buttonB.onEvent(ButtonEvent.Click, function () {
    light.setAll(0x000000)
})
input.buttonsAB.onEvent(ButtonEvent.Click, function () {
    if (scene < 7) {
        scene += 1
    } else if (scene == 7) {
        scene = 0
    }
})
let scene = 0
scene = 0
forever(function () {
    light.setBrightness(255)
    if (scene == 0) {
        light.showRing(
            "red red red red red red red red red red"
        )
    } else if (scene == 1) {
        light.showRing(
            `orange orange orange orange orange orange orange orange orange orange`
        )
    } else if (scene == 2) {
        light.showRing(
            `yellow yellow yellow yellow yellow yellow yellow yellow yellow yellow`
        )
    } else if (scene == 3) {
        light.showRing(
            `yellow orange pink red purple yellow orange pink red purple`
        )
    } else if (scene == 4) {
        light.showRing(
            `blue white blue white blue blue white blue white blue`
        )
    } else if (scene == 5) {
        light.showRing(
            `red orange red yellow red red orange red yellow red`
        )
    } else if (scene == 6) {
        light.showRing(
            `blue pink blue pink blue blue pink blue pink blue`
        )
    } else if (scene == 7) {
        light.showRing(
            `blue red blue red blue blue red blue red blue`
        )
    }
})
